August 2024 update: Instagram changed the wording it uses for potentially AI-generated or AI-edited images. Instead of Made with AI, the platform began displaying the less definitive AI info label. The method below reflects how the feature behaved at the time of this update.

Instagram introduced a feature intended to indicate when an image uploaded to a Story or post may have been created or altered using artificial intelligence. These images can receive an AI info label.

The system has not always worked reliably. Genuine photographs have occasionally received the label even though they were not AI-generated, while some AI-generated images have not been identified. Once Instagram adds the label, removing it from the published post may not be possible.

Here is one step that may reduce the chance of a false label appearing on a genuine photograph.

How to reduce the chance of Instagram adding an “AI info” label

Instagram can read metadata embedded in uploaded image files. This metadata may contain information left behind by editing or image-generation software. Certain entries can potentially contribute to the image being identified as AI-related.

Removing the metadata before uploading the photograph may therefore reduce the chance of false identification. Be aware that this also removes useful information such as camera details, copyright data and potentially location information.

Option 1: Use an online metadata remover

You can use an online service that removes EXIF and other metadata from an image file.

I do not specifically endorse either service, but the following tools appeared to work when I tested the general process:

  • Pics.io Metadata Remover
  • IMGonline EXIF Remover

Remember that uploading a photograph to a third-party service means sharing the file with that provider. Check its privacy terms before using it for sensitive or confidential images.

Option 2: Create a new image document manually

Open the finished photograph in Photoshop, Paint or another image editor that allows you to create a new document. Select and copy the complete image, create a new document with the same dimensions, paste the pixels into it and export the new file.

In Photoshop on Windows, the basic sequence is Ctrl+A, Ctrl+C, Ctrl+N, Enter and Ctrl+V. On a Mac, replace Ctrl with Cmd.

This copies the visible pixels into a new document rather than continuing to use the original file. Depending on the software and export settings, the new image may contain substantially less metadata.

Check the exported file before uploading it. Some editing applications automatically add their own software information during export.

Is this a perfect solution?

Probably not. Metadata is only one possible signal, and Instagram can change how its labelling system works at any time.

For a personal post, the label may not matter. It can become more awkward, however, when you are publishing an Instagram collaboration post with a company or larger brand and a genuine photograph is incorrectly presented as AI-related.

For that reason, checking the exported file and removing unnecessary metadata may still be worth the additional step when the origin of the photograph needs to remain completely clear.
